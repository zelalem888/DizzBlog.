## Getting Started

### 1 Clone the repo
```bash
git clone https://github.com/Hope-Dies-Here/random-blog.git
```
### 2 Select folder
```
cd random-blog
```
### 3 Install the dependacies
```bash
npm install
``` 
### 4 Start the server
```bash
npm run dev
```
### 5 Finnaly

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
